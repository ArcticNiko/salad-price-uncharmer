# Compiling Salad Price Uncharmer
## Prerequisite
* A recent version of Node.js (not sure how recent however)
## Instructions
1. Clone this repository (git: `git clone https://github.com/ArcticNiko/salad-price-uncharmer.git`)
2. Download all modules with `npm i`
3. Use `npm run build` for a minified userscript, otherwise use `npm run build-greasyfork` for more human-readable code
4. Your build will be in the `dist` folder
